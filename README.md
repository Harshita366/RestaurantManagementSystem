# 🍽️ Restaurant Management System

A responsive Restaurant Management System developed using HTML, CSS, and JavaScript.

This project was originally developed during my second year of B.Tech CSE while learning frontend web development fundamentals. It was later refined and published on GitHub in my third year to showcase my learning progress and project work. It demonstrates fundamental frontend development skills including UI structuring, styling, and interactive elements.
